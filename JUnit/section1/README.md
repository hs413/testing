# JUnit 테스트 만들기

## 단위 테스트 작성 이유
- 코드가 어떤 일을 하는지 알기 쉬움
- 단위 테스트가 쌓이면 회귀 테스트를 지원

### 체크리스트
- 추가적인 테스트를 작성할 필요가 있는가
- 결함이나 한계를 드러낼 수 있는 테스트를 작성할 수 있는가
